o que e um container? 
Diferente de uma vm, o container compartilha a kernel e a mesma arquitetura do SO.
esse processo permite rodar imagens(softwares) de uma forma isoalda do restante do PC.

O que sao imagens no container?
Um conjunto de softwares necessarios para realizar determinada operacao.

O que e um docker? (PaaS?)
E um servico que fornece diversas funcionalidades para o container. 
Equivalente ao git(aplicacao) > git hub(servico da aplicacao), exite o 
container(aplicacao) > docker hub(servico da apliacao). Fornece tambem maneiras de 
construir containers de forma ordenada e customizada com o docker file e docker compose.

O que e o docker file?
Equivalente ao Makefile do C, o docker file fornece instrucoes para construir a imagem
Docker, manusenado dependenciads, scripts de criacao, configuracoes de rede e etc.

O que e um docker composer?
Diferente do docker file que possui a receita para criar apenas um container, 
o docker compose permite configurar acriacao de multiplos containers e gerencia-los a
partir de um arquivo YAML. Ele permite configuracoes adicionais como a orquestracao
da inicializacao dos containers, conexoes entre eles, dependencias e etc.
